# Computational-Intelligence
This repository contains the code of the 2 exercises assigned in the mye-035 Computational Intelligence course.

Exercise 1 is a classic classification problem based on the multilayered Perceptron. 

  -Dataset1_soufle.java
  
  -MLP.java

Exercise 2 is a grouping exercise based on the k-means score.

  -Dataset2.java
  
  -GroupingProgram.java
  
  -MakeErrorPlot.java
  
  -MakePlot.java
  
  -Point.java
